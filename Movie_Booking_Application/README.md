A complete movie ticket booking system with secure authentication and admin management features.

## ✨ Features
- **User Authentication** (JWT-based registration/login)
- **Theater Management** (CRUD operations)
- **Show Scheduling** with seat availability
- **Booking System** with ticket generation
- **Admin Dashboard** for content management

## 🛠 Tech Stack
- **Backend**: Spring Boot 3.1, Spring Security
- **Database**: MySQL 8.0
- **Authentication**: JWT
- **API Documentation**: Swagger UI
